# ☕ Daily Brew – Coffee Shop Landing Page

A clean and modern landing page designed for a fictional local coffee shop called **Daily Brew**.

This project was built using **HTML and CSS only**, with a focus on clean layout structure, warm coffee-inspired colors, modern card design, and smooth hover interactions.

---

## ✨ Features
- Responsive navigation bar
- Hero section with call-to-action button
- Warm and inviting coffee-shop color palette
- About section with friendly brand message
- Menu cards for featured drinks
- Contact information section
- Footer section
- Hover effects on cards and buttons
- Mobile responsive layout

---

## 🛠 Tech Stack
- HTML5
- CSS3
- Flexbox
- Media Queries

---

## 🎨 Design Theme
Inspired by cozy local cafés using:
- Cream
- Brown
- Beige
- Light caramel
